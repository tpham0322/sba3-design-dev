# Frontend Mentor - Chat App CSS Illustration

A responsive solution to the **Chat App CSS Illustration** challenge from Frontend Mentor. This project recreates a modern chat interface using **HTML5**, **CSS3**, and **Bootstrap 5**.

## Overview

This project is my solution to the **Chat App CSS Illustration** challenge from Frontend Mentor. The goal was to recreate the provided design while practicing responsive web development using HTML, CSS, and Bootstrap.

Bootstrap was used to build the responsive layout and reusable UI components, while custom CSS was added to closely match the original design.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Google Fonts

## Features

- Responsive layout for desktop and mobile devices
- Bootstrap Grid System
- Bootstrap Card component
- Bootstrap Button and Input Group
- Bootstrap utility classes for spacing and alignment
- Custom CSS for chat bubbles, gradients, and phone styling

## My Approach

I started by creating the overall page layout using Bootstrap's grid system. The layout is divided into two columns: one containing the phone interface and another containing the project description.

After building the layout, I created the phone mockup using Bootstrap components and custom CSS. Since Bootstrap does not include chat bubbles or decorative background shapes, I used custom CSS to recreate the original Frontend Mentor design.

Finally, I added responsive media queries to ensure the layout works well on both desktop and mobile devices.

## Challenges

One of the biggest challenges was combining Bootstrap with a highly customized design. While Bootstrap made the layout responsive, the phone interface, gradient backgrounds, and chat bubbles required custom CSS. I solved this by using Bootstrap for the structure and custom CSS for the detailed styling.

## What I Learned

This project helped me improve my skills in:

- Responsive web design
- Bootstrap Grid System
- Bootstrap utility classes
- Flexbox
- Custom CSS styling
- Recreating UI designs from mockups
